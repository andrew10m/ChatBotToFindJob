# Умный конструктор физических моделей

Это веб-приложение позволяет пользователям вводить текстовое описание простой физической задачи на русском языке, парсит текст и автоматически генерирует 2D-диаграмму, визуализирующую задачу. Пользователи также могут изменять параметры (например, массу, угол) и видеть изменения в визуализации.

## Технологии

*   **Фронтенд:** HTML, CSS, JavaScript, p5.js (для визуализации)
*   **Бэкенд:** Python, Flask (для обработки текста и API)

## Структура проекта

```
.
├── app.py             # Flask бэкенд-приложение
├── index.html         # Основная HTML страница
├── script.js          # JavaScript для фронтенда и логики p5.js
├── styles.css         # Стили для HTML страницы
├── requirements.txt   # Python зависимости
└── README.md          # Этот файл
```

## Запуск приложения локально

1.  **Клонируйте репозиторий (если он уже на GitHub) или скачайте файлы проекта.**

2.  **Убедитесь, что у вас установлен Python 3.**

3.  **Создайте и активируйте виртуальное окружение (рекомендуется):**
    ```bash
    python -m venv venv
    # Для Windows
    venv\Scripts\activate
    # Для macOS/Linux
    source venv/bin/activate
    ```

4.  **Установите зависимости Python:**
    ```bash
    pip install -r requirements.txt
    ```

5.  **Запустите Flask-приложение (бэкенд):**
    Откройте терминал в корневой директории проекта и выполните:
    ```bash
    python app.py
    ```
    Вы должны увидеть сообщение, что сервер запущен, обычно по адресу `http://127.0.0.1:5000/`.

6.  **Откройте фронтенд в браузере:**
    Откройте файл `index.html` в вашем веб-браузере. Вы можете просто дважды щелкнуть по файлу `index.html` в проводнике.

7.  **Использование:**
    *   Введите описание физической задачи в текстовое поле (например, "Брусок массой 2 кг на наклонной плоскости с углом 30 градусов" или "Тело брошено под углом 45 градусов со скоростью 10 м/с").
    *   Нажмите кнопку "Обработать".
    *   Параметры должны извлечься и отобразиться в соответствующих полях, а также должна появиться визуализация.
    *   Вы можете изменять параметры в полях ввода, и визуализация будет обновляться автоматически.

## Загрузка кода на GitHub

1.  **Установите Git**, если он у вас еще не установлен. ([https://git-scm.com/downloads](https://git-scm.com/downloads))

2.  **Создайте новый репозиторий на GitHub.**
    *   Перейдите на [GitHub](https://github.com/) и войдите в свой аккаунт.
    *   Нажмите кнопку "New" (Новый) или "+" в правом верхнем углу и выберите "New repository".
    *   Дайте вашему репозиторию имя (например, `physics-model-constructor`).
    *   Вы можете добавить описание.
    *   Выберите, будет ли репозиторий публичным или приватным.
    *   **Не инициализируйте** репозиторий с README, .gitignore или лицензией, если вы уже создали эти файлы локально.
    *   Нажмите "Create repository".

3.  **Инициализируйте Git в локальной директории проекта:**
    Откройте терминал или командную строку в корневой папке вашего проекта и выполните следующие команды:
    ```bash
    git init -b main  # Инициализирует новый репозиторий с веткой 'main'
    git add .         # Добавляет все файлы проекта в индекс для коммита
    git commit -m "Initial commit: Basic structure and functionality"
                      # Создает первый коммит с сообщением
    ```

4.  **Свяжите локальный репозиторий с удаленным репозиторием на GitHub:**
    Скопируйте URL вашего нового репозитория на GitHub (он будет выглядеть примерно так: `https://github.com/YourUsername/YourRepositoryName.git`).
    Выполните команду:
    ```bash
    git remote add origin https://github.com/YourUsername/YourRepositoryName.git
                      # Замените URL на свой
    ```

5.  **Отправьте (push) ваши локальные коммиты на GitHub:**
    ```bash
    git push -u origin main
    ```
    Эта команда отправит вашу ветку `main` на GitHub и установит ее как вышестоящую ветку по умолчанию.

Теперь ваш код должен быть загружен на GitHub. Для последующих изменений используйте:
```bash
git add .
git commit -m "Сообщение о ваших изменениях"
git push origin main
```
